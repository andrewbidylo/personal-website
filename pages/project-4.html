Interview Scheduler

GitHub link

Interview Scheduler is a single-page applaction. Front-end: React, JavaScript, HTML, CSS, SASS, AJAX. Back-end: Node JS, Express. Database: PostgreSQL. Testing: Storybook, Jest, Cypress.

As a user you can:

- add/remove products to a cart

- see products in a cart

- mark/unmark favorited items

- see favorited items

- see price for added to cart items in the header and in the cart

- search itmes by name

- buy items

- see ordered items

Developmet

For development I have chosen React and started with the main functionality and reusable component then. Then step-by-step I added new features.

I created fake data on the mockapi.io. To comunicate with back-end and making reqestes I chose AXIOS library and asinc/await.

During developing I used useEffect, useState, useContext and own custom hook.
