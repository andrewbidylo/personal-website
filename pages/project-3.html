About the project

Sourse code

Tools used:

  • JavaScript
  • HTML
  • CSS
  • Figma

Before starting to develop this project, I set up a few tasks for myself:

First, the website should have functionality that allowed users to easily find all information about my including social media and GitHub.

Second, everyone should be able to download my current resume and take a look at my projects.

Third, I decided to wrap it up in a nice and responsive design.

Project design

Design for this project I created in Figma. The main purpose was that design should be responsible and nice. So, I chose a few colors for design - black and crimson (which look pretty nice in my opinion 😜), and a few fonts - Poppins and Ubuntu. After that, using Figma I made a mobile and desktop version of design. And finally Implemented this design into the project.

Developmet

The main part of the website was built by using HTML and CSS, but next functionality was added by JavaScript:

- changing the colours of the header

- using the scroll button

- dropdown menu

- sliders

- submit form
